# 03-应用开发

Web 前后端开发与桌面应用开发的入门知识。

## 目录

| 编号 | 文件 | 内容 |
|------|------|------|
| (1) | 应用开发概述 | Web/桌面/移动端三大方向对比、学习路径建议 |
| (2) | 前端基础 | HTML/CSS/JS、前端框架（Vue/React）、浏览器与原生应用区别 |
| (3) | 后端基础 | 服务端职责、技术栈对比、Flask示例 |
| (4) | 前后端协作 | 请求-响应流程、DNS解析、全栈学习路线 |
| (5) | Qt框架与跨平台开发 | 信号与槽机制、Qt5 vs Qt6、Widgets vs QML |
| (6) | WinUI3与Windows现代开发 | Windows App SDK、WinRT架构、C++/WinRT vs C# |

## 学习路径

- **Web全栈**：(1) → (2) → (3) → (4)
- **桌面开发**：(1) → (5) Qt 或 (6) WinUI3
