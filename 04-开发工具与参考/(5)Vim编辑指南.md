## Vim 是什么

Vim 是 Linux/Unix 系统标配的终端文本编辑器，几乎所有服务器都预装。它的核心特点是**纯键盘操作**，熟练后编辑效率远超普通编辑器。

> **Vi vs Vim**：Vim（Vi IMproved）是 Vi 的增强版，支持语法高亮、多级撤销、插件系统等。现代 Linux 发行版通常默认安装 Vim。

### 为什么学 Vim

- 服务器运维必备（SSH 连上服务器，能用的可能只有 Vi/Vim）
- 不依赖鼠标，手不离键盘效率高
- 几乎所有 IDE/编辑器都有 Vim 插件（VS Code、JetBrains、甚至 Word）

## 四种模式

Vim 的核心设计：不同模式下按键含义完全不同。

| 模式 | 进入方式 | 作用 | 退出方式 |
|------|----------|------|----------|
| **Normal（普通模式）** | 按 `Esc` | 移动光标、删除、复制等操作 | — |
| **Insert（插入模式）** | 按 `i`、`a`、`o` 等 | 输入文本内容 | `Esc` |
| **Command（命令模式）** | 按 `:` | 执行保存、退出、搜索等命令 | `Enter` 执行 / `Esc` 取消 |
| **Visual（可视模式）** | 按 `v`、`V`、`Ctrl+v` | 选中文本，再执行操作 | `Esc` |

> **核心记忆**：不确定当前是什么模式？按 `Esc` 回到 Normal 模式。

## 启动与退出

### 打开文件

```bash
vim                    # 打开空文件
vim file.txt           # 打开指定文件
vim +10 file.txt       # 打开并跳到第 10 行
vim -R file.txt        # 只读模式打开
```

### 退出（在 Normal 模式下按 `:` 进入命令模式）

| 命令 | 作用 |
|------|------|
| `:q` | 退出（未修改时） |
| `:q!` | 强制退出，丢弃修改 |
| `:w` | 保存 |
| `:wq` | 保存并退出 |
| `:wq!` | 强制保存并退出 |
| `:x` | 保存并退出（同 `:wq`） |
| `ZZ` | Normal 模式下直接保存退出 |
| `ZQ` | Normal 模式下直接退出不保存 |

## 编辑操作

Vim 的操作遵循 **动词 + 范围** 的语法：

```
d + w  = 删除一个单词
c + i + " = 修改引号内的内容
y + $  = 复制到行尾
```

### 进入插入模式

| 按键 | 作用 |
|------|------|
| `i` | 在光标前插入 |
| `a` | 在光标后插入 |
| `I` | 在行首插入 |
| `A` | 在行尾插入 |
| `o` | 在下方新开一行并插入 |
| `O` | 在上方新开一行并插入 |

### 删除

| 按键 | 作用 |
|------|------|
| `x` | 删除光标下的字符 |
| `dd` | 删除整行 |
| `dw` | 删除一个单词 |
| `d$` 或 `D` | 删除到行尾 |
| `d0` | 删除到行首 |
| `dG` | 删除到文件末尾 |
| `dgg` | 删除到文件开头 |
| `3dd` | 删除 3 行 |

### 复制与粘贴

| 按键 | 作用 |
|------|------|
| `yy` | 复制整行 |
| `yw` | 复制一个单词 |
| `y$` | 复制到行尾 |
| `3yy` | 复制 3 行 |
| `p` | 在光标后粘贴 |
| `P` | 在光标前粘贴 |

### 撤销与重做

| 按键 | 作用 |
|------|------|
| `u` | 撤销 |
| `Ctrl+r` | 重做 |
| `.` | 重复上一个操作（非常实用） |

### 修改（删除并进入插入模式）

| 按键 | 作用 |
|------|------|
| `cw` | 修改一个单词 |
| `cc` | 修改整行 |
| `C` | 修改到行尾 |
| `ci"` | 修改引号内的内容 |
| `ci(` | 修改括号内的内容 |
| `ciw` | 修改光标所在的单词 |

## 查找与替换

### 查找

| 命令 | 作用 |
|------|------|
| `/keyword` | 向下查找 keyword |
| `?keyword` | 向上查找 keyword |
| `n` | 跳到下一个匹配 |
| `N` | 跳到上一个匹配 |
| `*` | 查找光标下的单词（向下） |
| `#` | 查找光标下的单词（向上） |

### 替换

```
:[范围]s/旧文本/新文本/[标志]
```

| 命令 | 作用 |
|------|------|
| `:s/old/new/` | 替换当前行第一个匹配 |
| `:s/old/new/g` | 替换当前行所有匹配 |
| `:%s/old/new/g` | 替换全文所有匹配 |
| `:%s/old/new/gc` | 替换全文，每次确认 |
| `:10,20s/old/new/g` | 替换第 10-20 行 |

> `%` 表示全文，`g` 表示每行所有匹配，`c` 表示逐个确认。

## 实用技巧

### 重复操作

`.`（点命令）是 Vim 最强大的功能之一——重复上一次编辑操作。

```
# 场景：把多行末尾加分号
1. 光标到第一行末尾 → A;Esc    （行尾加分号）
2. j. j. j.                      （下一行，重复）
```

### 宏录制

批量重复复杂操作：

```
qa       开始录制到寄存器 a
...      执行一系列操作
q        停止录制
@a       执行宏 a
10@a     执行宏 a 共 10 次
```

### 标记

| 命令 | 作用 |
|------|------|
| `ma` | 在当前位置设置标记 a |
| `` `a `` | 跳到标记 a 的位置 |
| `:marks` | 列出所有标记 |

### 寄存器

```bash
"ayy     复制整行到寄存器 a
"ap      粘贴寄存器 a 的内容
"+y      复制到系统剪贴板
"+p      从系统剪贴板粘贴
```

### 执行 Shell 命令

```
:!ls            执行 shell 命令
:!gcc % -o out  编译当前文件
:r !date        将 date 命令输出插入到当前位置
```

## Vim 配置

### 配置文件位置

| 文件 | 作用 |
|------|------|
| `~/.vimrc` | 用户配置（推荐） |
| `/etc/vim/vimrc` | 全局配置 |

### 推荐基础配置

```vim
" ~/.vimrc

set number              " 显示行号
set relativenumber      " 相对行号
set tabstop=4           " Tab 宽度 4
set shiftwidth=4        " 缩进宽度 4
set expandtab           " Tab 转空格
set autoindent          " 自动缩进
set smartindent         " 智能缩进
set hlsearch            " 高亮搜索结果
set incsearch           " 增量搜索
set ignorecase          " 搜索忽略大小写
set smartcase           " 有大写时区分大小写
set cursorline          " 高亮当前行
set showmatch           " 显示匹配括号
set encoding=utf-8      " UTF-8 编码
syntax on               " 语法高亮
set mouse=a             " 启用鼠标支持
set clipboard=unnamedplus " 使用系统剪贴板
```
