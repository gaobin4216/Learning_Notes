# 应用开发概述

> **三种方向，选一个深入，其他了解即可。**

---

## 一、应用开发的三大方向

| 方向 | 做什么 | 用什么技术 | 适合谁 |
|------|--------|-----------|--------|
| **网页开发** | 网站、Web应用、小程序 | HTML/CSS/JS + 后端语言 | 想快速上手、做产品、全栈开发 |
| **桌面应用开发** | Windows/Mac/Linux软件 | Qt(C++)、WinUI3(C#)、Electron | 想做原生软件、工具类应用 |
| **移动应用开发** | 手机APP | Flutter、React Native、原生(iOS/Android) | 想做手机应用 |

## 三、桌面应用开发技术对比

本目录重点介绍两种Windows桌面开发技术：

| 对比 | Qt | WinUI3 |
|------|-----|--------|
| **语言** | C++ | C# |
| **跨平台** | ✅ Windows/Mac/Linux/嵌入式 | ❌ 仅Windows |
| **UI风格** | 传统+现代(QML) | Windows 11 Fluent Design |
| **学习难度** | ⭐⭐⭐ (C++门槛高) | ⭐⭐ (C#更易上手) |
| **适用场景** | 跨平台软件、工业软件、嵌入式 | Windows桌面应用、微软生态 |
| **构建工具** | CMake | MSBuild/NuGet |

**选择建议：**
- 想跨平台 → Qt
- 只做Windows且想用微软最新技术 → WinUI3
- 新手想快速上手桌面开发 → 先学C#再学WinUI3

## 四、新手学习路线建议

### 如果你想做网页/Web应用（推荐新手）

```
(2)前端基础.md → (3)后端基础.md → (4)前后端协作.md
```

**优势**：入门快、岗位多、成果可见（浏览器直接看效果）

### 如果你想做桌面软件

```
(5)Qt跨平台开发.md 或 (6)WinUI3-Windows开发.md
```

**优势**：性能好、可离线、适合工具类软件

### 学习顺序建议

| 阶段 | 学什么 | 时间 |
|------|--------|------|
| 第1阶段 | 网页前端基础（HTML/CSS/JS） | 1-2个月 |
| 第2阶段 | 选一个方向深入 | 3-6个月 |
| 第3阶段 | 做项目练手 | 持续 |

## 五、常见问题

| 问题 | 回答 |
|------|------|
| 前端和后端学哪个？ | 看兴趣：喜欢界面→前端，喜欢逻辑→后端，都想学→全栈 |
| Qt和WinUI3选哪个？ | 跨平台→Qt，只做Windows→WinUI3 |
| 网页和桌面应用学哪个？ | 网页入门快、岗位多；桌面应用性能好、适合工具软件 |
| 学编程一定要学前端吗？ | 不一定，看你的目标方向 |

---

> **总结**：网页开发适合快速上手和全栈方向，桌面开发适合做原生工具软件。新手建议先从前端入门，再根据兴趣选择深入方向。
