# 工具箱、包管理与外部库集成

## 一、工具箱（Toolbox）

工具箱是 MathWorks 官方发布的专业领域扩展包，包含函数、App 和文档。

### 常用工具箱

| 工具箱 | 功能 |
|--------|------|
| **Signal Processing** | 信号处理、滤波、频谱分析 |
| **Image Processing** | 图像处理与分析 |
| **Control System** | 控制系统设计与仿真 |
| **Statistics and Machine Learning** | 统计分析、机器学习 |
| **Deep Learning** | 深度学习模型训练 |
| **Optimization** | 优化问题求解 |
| **Symbolic Math** | 符号计算 |
| **Simulink** | 图形化仿真环境 |

### 查看与安装

```matlab
ver                           % 查看已安装的工具箱
license('test', 'Signal_Toolbox')  % 检查某个工具箱是否可用
toolboxdir('signal')          % 查看工具箱安装路径
```

安装方式：`主页 → 附加功能 → 获取附加功能`，或从 MathWorks 官网下载。

## 二、File Exchange（社区代码）

[MathWorks File Exchange](https://www.mathworks.com/matlabcentral/fileexchange/) 是社区代码分享平台，免费下载各类 MATLAB 函数和工具。

### 使用流程

1. 下载代码（ZIP）
2. 解压到项目目录
3. 添加到 MATLAB 路径

```matlab
addpath('下载的文件夹路径');   % 临时添加
addpath(genpath('my_project')); % 递归添加子目录
savepath;                       % 永久保存路径
```

## 三、路径管理

MATLAB 通过**搜索路径**查找函数文件，路径管理是日常开发的基础。

| 函数 | 作用 |
|------|------|
| `path` | 查看当前搜索路径 |
| `addpath` | 添加路径 |
| `rmpath` | 移除路径 |
| `genpath` | 生成包含所有子目录的路径字符串 |
| `savepath` | 永久保存路径修改 |
| `restoredefaultpath` | 恢复默认路径 |

## 四、外部语言互操作

### Python 互操作

```matlab
% 调用 Python
result = py.math.sqrt(16);
np = py.importlib.import_module('numpy');
arr = np.array({1, 2, 3});

% 设置 Python 版本
pyenv('Version', 'C:/Python312/python.exe');
```

### Java 互操作

MATLAB 内置 JVM，可直接使用 Java 类库：

```matlab
list = java.util.ArrayList();
list.add('hello');

javaaddpath('第三方库.jar');  % 添加第三方 JAR
```

### C/C++ MEX 文件

用 C/C++ 编写高性能函数，在 MATLAB 中直接调用：

```matlab
mex myadd.c    % 编译
result = myadd(3, 4);  % 调用
```

## 五、项目依赖管理建议

```
myproject/
├── src/              % 源代码
├── lib/              % 第三方库（File Exchange 等）
├── data/             % 数据文件
├── tests/            % 测试脚本
├── setup_paths.m     % 路径初始化
└── README.md
```

```matlab
% setup_paths.m — 团成员 clone 后运行一次即可
function setup_paths()
    root = fileparts(mfilename('fullpath'));
    addpath(genpath(fullfile(root, 'src')));
    addpath(genpath(fullfile(root, 'lib')));
end
```
