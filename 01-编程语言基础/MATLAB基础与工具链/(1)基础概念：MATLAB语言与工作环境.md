# MATLAB 语言与工作环境

## 一、MATLAB 语言概述

MATLAB（Matrix Laboratory）是由 MathWorks 公司开发的商业数学软件，诞生于 1984 年。它集数值计算、可视化和编程于一体，广泛应用于工程计算、控制设计、信号处理、图像处理等领域。

### MATLAB 的特点

| 特点 | 说明 |
|------|------|
| **矩阵运算为核心** | 所有数据以矩阵形式存储和操作 |
| **解释型** | 逐行执行，无需编译 |
| **丰富的工具箱** | 针对各专业领域的扩展包 |
| **强大的绘图** | 内置高质量数据可视化 |
| **交互式环境** | 命令行即时执行 |
| **跨平台** | 支持 Windows、macOS、Linux |

### 适用领域

- 数值分析与科学计算
- 控制系统设计与仿真（Simulink）
- 信号与图像处理
- 机器学习与深度学习
- 通信系统设计
- 金融建模与量化分析

## 二、MATLAB 的运行机制

### 执行方式

```
MATLAB 代码(.m) → MATLAB 解释器 → JIT 编译 → 执行
```

MATLAB 是解释型语言，但内置 JIT（Just-In-Time）加速器，对循环密集型代码会自动编译优化。

### 两种 M 文件类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **脚本文件** | 一系列命令的集合，共享工作区变量 | `myscript.m` |
| **函数文件** | 有独立输入输出，使用局部变量 | `myfunction.m` |

```matlab
% 脚本文件示例：script_demo.m
x = 1:10;
y = x.^2;
plot(x, y);
title('Script Demo');
```

```matlab
% 函数文件示例：myadd.m
function result = myadd(a, b)
    result = a + b;
end
```

## 三、MATLAB 数据类型

### 基本数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **double** | 双精度浮点（默认） | `x = 3.14` |
| **single** | 单精度浮点 | `x = single(3.14)` |
| **int8/16/32/64** | 有符号整数 | `x = int32(42)` |
| **uint8/16/32/64** | 无符号整数 | `x = uint8(255)` |
| **logical** | 逻辑值 | `x = true` |
| **char** | 字符 | `x = 'hello'` |
| **string** | 字符串（R2016b+） | `x = "hello"` |

### 矩阵与数组

MATLAB 中一切皆矩阵，标量和向量是矩阵的特例。

```matlab
% 标量
a = 5;

% 行向量
v = [1, 2, 3, 4, 5];

% 列向量
v = [1; 2; 3; 4; 5];

% 矩阵
M = [1, 2, 3;
     4, 5, 6;
     7, 8, 9];

% 常用创建方式
z = zeros(3, 4);     % 3x4 全零矩阵
o = ones(3, 4);      % 3x4 全一矩阵
e = eye(3);          % 3x3 单位矩阵
r = rand(3, 4);      % 3x4 随机矩阵
s = linspace(0, 1, 100);  % 等间距向量
```

### 矩阵运算

```matlab
A = [1, 2; 3, 4];
B = [5, 6; 7, 8];

% 矩阵运算
C = A + B;       % 加法
D = A * B;       % 矩阵乘法
E = A';          % 转置
F = inv(A);      % 求逆
G = det(A);      % 行列式

% 元素级运算（加点号）
H = A .* B;      % 元素乘法
I = A ./ B;      % 元素除法
J = A .^ 2;      % 元素幂
```

## 四、MATLAB 版本与发布

### 版本命名

MATLAB 使用年份+版本号命名，每年发布两个版本（a 和 b）：

| 版本 | 发布年份 | 重要特性 |
|------|----------|----------|
| **R2014b** | 2014 | 新图形系统 |
| **R2016b** | 2016 | string 类型、实时编辑器 |
| **R2018b** | 2018 | 深度学习工具箱增强 |
| **R2020a** | 2020 | 深度学习设计器 |
| **R2022a** | 2022 | Python 接口改进 |
| **R2024a** | 2024 | AI 增强功能 |

> **建议**：使用学校或单位提供的最新版本，个人学习也可使用 MATLAB Online。

## 五、MATLAB 的执行方式

### 1. 命令窗口（交互式）

```matlab
>> x = 1:5
x = 1 2 3 4 5

>> mean(x)
ans = 3

>> help mean
% 显示 mean 函数的帮助文档
```

### 2. 脚本执行

```matlab
% 在编辑器中编写脚本，按 F5 运行
% 或在命令窗口输入脚本名
>> myscript
```

### 3. 实时脚本（Live Script）

`.mlx` 文件，支持富文本、公式、图片和代码混排，类似 Jupyter Notebook。

### 4. 函数调用

```matlab
>> result = myadd(3, 4)
result = 7
```
