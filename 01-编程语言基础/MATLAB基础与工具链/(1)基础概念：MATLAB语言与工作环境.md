# MATLAB 语言与工作环境

## 一、MATLAB 是什么？

MATLAB（Matrix Laboratory，矩阵实验室）是 MathWorks 公司开发的商业数学软件，诞生于 1984 年。它的核心设计理念是**一切皆矩阵**——所有数据都以矩阵形式存储和操作，这让数学和工程计算变得非常直观。

### MATLAB 与其他数学工具的区别

| 工具 | 定位 | 特点 |
|------|------|------|
| **MATLAB** | 商业数学软件 | 功能最全，工具箱生态最丰富，需要付费 |
| **Python + NumPy/SciPy** | 开源科学计算 | 免费，社区活跃，但需要自己组合库 |
| **Mathematica** | 符号计算 | 符号推导能力强，数值计算不如 MATLAB |
| **R** | 统计分析 | 统计领域最强，工程计算不如 MATLAB |
| **Julia** | 高性能科学计算 | 语法类似 MATLAB，速度接近 C |

### MATLAB 的核心优势

| 优势 | 说明 |
|------|------|
| **矩阵运算为原生操作** | 矩阵加减乘除、求逆、特征值等都是内置操作 |
| **工具箱生态** | 信号处理、控制系统、深度学习等 100+ 专业工具箱 |
| **强大的绘图** | 一行代码出高质量图形，交互式数据探索 |
| **Simulink** | 图形化建模仿真，工程领域独一无二 |
| **代码生成** | MATLAB 代码可自动生成 C/C++ 代码，部署到嵌入式设备 |
| **文档与示例** | 每个函数都有完善的文档和示例代码 |

### 适用领域

- 数值分析与科学计算
- 控制系统设计与仿真（Simulink）
- 信号与图像处理
- 机器学习与深度学习
- 通信系统设计
- 金融建模与量化分析
- 航空航天与汽车工程
- 生物医学工程

## 二、MATLAB 的运行机制

### 执行方式

```
MATLAB 代码(.m)
    ↓  MATLAB 解释器逐行读取
JIT 加速器（对热点代码自动编译优化）
    ↓
执行并输出结果
```

MATLAB 是解释型语言，但内置 **JIT（Just-In-Time）加速器**，对循环密集型代码会自动编译优化，性能接近编译型语言。

### 两种 M 文件

| 类型 | 特点 | 用途 |
|------|------|------|
| **脚本文件（Script）** | 一系列命令的集合，共享工作区变量 | 数据分析流程、批处理 |
| **函数文件（Function）** | 有独立输入输出，使用局部变量 | 封装可复用的逻辑 |

> **关键区别**：脚本文件直接操作工作区变量，函数文件有自己的局部变量空间，不会污染工作区。

### 工作区（Workspace）

工作区是 MATLAB 的核心概念之一：

```
命令窗口输入 x = 1:10
    ↓
变量 x 存入工作区（内存中）
    ↓
工作区窗口可以查看 x 的值、大小、类型
    ↓
脚本和命令窗口共享同一个工作区
    ↓
clear 命令可以清除工作区中的变量
```

## 三、MATLAB 版本

### 版本命名规则

MATLAB 使用 **年份 + 版本号** 命名，每年发布两个版本：
- **a 版**：上半年发布（3 月左右）
- **b 版**：下半年发布（9 月左右）

例如：`R2024a` = 2024 年上半年发布的版本。

### 主要版本

| 版本 | 发布年份 | 重要特性 |
|------|----------|----------|
| **R2014b** | 2014 | 新图形系统（HG2） |
| **R2016b** | 2016 | string 类型、实时编辑器 |
| **R2018b** | 2018 | 深度学习工具箱增强 |
| **R2020a** | 2020 | 深度学习设计器 |
| **R2022a** | 2022 | Python 接口改进 |
| **R2024a** | 2024 | AI 增强功能 |

> **建议**：使用学校或单位提供的最新版本。如果没有本地许可证，可以使用 MATLAB Online。

### 许可证类型

| 类型 | 说明 | 适用人群 |
|------|------|----------|
| **校园许可证** | 学校统一购买，学生免费 | 在校学生 |
| **学术版** | 价格较低 | 高校教师、研究人员 |
| **商业版** | 按年付费 | 企业用户 |
| **个人版** | 价格较高 | 个人开发者 |
| **MATLAB Online** | 有许可证即可在线使用 | 所有许可证持有者 |

## 四、MATLAB 的执行方式

### 1. 命令窗口（交互式）

最直接的使用方式，在命令窗口逐行输入命令：

```matlab
>> x = 1:5
x = 1 2 3 4 5

>> mean(x)
ans = 3

>> help mean
% 显示 mean 函数的帮助文档
```

适用场景：快速计算、探索数据、测试函数。

### 2. 脚本执行

在编辑器中编写 `.m` 文件，按 `F5` 运行。脚本中的命令依次执行，共享工作区变量。

适用场景：可重复的数据分析流程、批处理操作。

### 3. 实时脚本（Live Script）

`.mlx` 文件，支持富文本、公式、图片和代码混排，类似 Jupyter Notebook。

适用场景：教学演示、实验报告、交互式数据分析。

### 4. 函数调用

函数文件可以被其他脚本或函数调用，实现代码复用。

适用场景：封装可复用的算法、构建工具库。

## 五、MATLAB 的搜索路径

### 什么是搜索路径？

当你在命令窗口输入一个函数名（如 `plot`），MATLAB 需要知道去哪里找这个函数的 `.m` 文件。**搜索路径**就是 MATLAB 查找函数文件的目录列表。

### 路径管理命令

| 命令 | 作用 |
|------|------|
| `path` | 查看当前搜索路径 |
| `addpath('dir')` | 添加目录到搜索路径 |
| `addpath(genpath('dir'))` | 递归添加目录及其所有子目录 |
| `rmpath('dir')` | 从搜索路径中移除目录 |
| `savepath` | 永久保存路径修改 |
| `restoredefaultpath` | 恢复到默认搜索路径 |

### 路径冲突

如果两个目录下有同名的 `.m` 文件，MATLAB 会使用**搜索路径中靠前的那个**。可以用 `which funcname` 查看实际使用的是哪个文件。
