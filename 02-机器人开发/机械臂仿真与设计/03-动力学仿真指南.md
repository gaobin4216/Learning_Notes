## 1. 动力学基础概念

### 1.1 动力学研究什么？

```
动力学（Dynamics）：研究力/力矩与运动之间的关系

核心问题：
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  正向动力学（Forward Dynamics）：                            │
│  ─────────────────────────────                              │
│  已知：关节力矩 τ                                            │
│  求解：关节加速度 q̈                                          │
│                                                             │
│  应用：仿真（给定力矩，预测运动）                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  逆向动力学（Inverse Dynamics）：                            │
│  ─────────────────────────────                              │
│  已知：期望的运动轨迹 q, q̇, q̈                                │
│  求解：所需关节力矩 τ                                        │
│                                                             │
│  应用：控制（计算实现期望运动所需的力矩）                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 动力学方程的一般形式

```
标准形式：
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  M(q)q̈ + C(q,q̇)q̇ + G(q) = τ + τ_ext                       │
│                                                             │
│  其中：                                                      │
│  - M(q)：惯性矩阵（7×7 对称正定）                           │
│  - C(q,q̇)：科里奥利力和离心力矩阵                           │
│  - G(q)：重力向量（7×1）                                    │
│  - τ：关节驱动力矩（7×1）                                   │
│  - τ_ext：外部力矩（如末端负载）（7×1）                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘

各项的物理含义：

1. M(q)q̈ —— 惯性力项
   - 描述加速运动所需的力
   - M(q)是关节角度的函数（构型相关）
   - 对称正定矩阵

2. C(q,q̇)q̇ —— 科里奥利力和离心力项
   - 旋转运动产生的耦合力
   - 与关节速度的平方成正比
   - 包含科里奥利力和离心力

3. G(q) —— 重力项
   - 克服重力所需的力
   - 与关节角度相关（构型相关）

4. τ_ext —— 外部力矩
   - 末端负载
   - 环境接触力
```

### 1.3 动力学方程的物理图像

```
简化理解（以单关节为例）：

τ = I × α + m × g × L × sin(θ)
    ↑              ↑
    惯性力         重力

其中：
- I：转动惯量
- α：角加速度
- m：质量
- g：重力加速度
- L：质心到关节的距离
- θ：关节角度

对于多关节系统，还需要考虑：
- 关节之间的耦合（科里奥利力）
- 离心力
```

## 2. 拉格朗日动力学推导

### 2.1 拉格朗日方程

```
拉格朗日方程（Lagrange's Equation）：

d/dt(∂L/∂q̇) - ∂L/∂q = τ

其中：
- L = K - P（拉格朗日量）
- K：系统总动能
- P：系统总势能
- q：广义坐标（关节角度）
- τ：广义力（关节力矩）

推导步骤：
┌─────────────────────────────────────────────────────────────┐
│  1. 确定广义坐标 q = [θ1, θ2, ..., θ7]                      │
│  2. 计算系统总动能 K                                         │
│  3. 计算系统总势能 P                                         │
│  4. 构造拉格朗日量 L = K - P                                 │
│  5. 代入拉格朗日方程求解                                     │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 动能计算

```
单连杆的动能：

K_i = 1/2 × m_i × v_ci^T × v_ci + 1/2 × ω_i^T × I_i × ω_i
                ↑                           ↑
           线性动能                      旋转动能

其中：
- m_i：连杆i的质量
- v_ci：连杆i质心的线速度
- ω_i：连杆i的角速度
- I_i：连杆i的惯性张量（3×3矩阵）

系统总动能：
K = Σ K_i = Σ [1/2 × m_i × v_ci^T × v_ci + 1/2 × ω_i^T × I_i × ω_i]
```

### 2.3 速度的雅可比表示

```
关键：将连杆速度用关节速度表示

v_ci = J_vi(q) × q̇
ω_i = J_ωi(q) × q̇

其中：
- J_vi：线速度雅可比（3×7）
- J_ωi：角速度雅可比（3×7）

代入动能公式：
K = 1/2 × q̇^T × M(q) × q̇

其中惯性矩阵：
M(q) = Σ [m_i × J_vi^T × J_vi + J_ωi^T × I_i × J_ωi]
```

### 2.4 惯性矩阵推导

```python
import numpy as np
from typing import List

class DynamicsModel:
    """七自由度机械臂动力学模型"""

    def __init__(self, kinematics, link_params):
        """
        参数：
            kinematics: 运动学对象
            link_params: 连杆参数
                mass: 各连杆质量 (7,)
                inertia: 各连杆惯性张量 (7,3,3)
                com: 各连杆质心位置（在连杆坐标系中）(7,3)
        """
        self.kin = kinematics
        self.mass = np.array(link_params['mass'])
        self.inertia = np.array(link_params['inertia'])
        self.com = np.array(link_params['com'])

    def compute_mass_matrix(self, q: np.ndarray) -> np.ndarray:
        """
        计算惯性矩阵 M(q)

        数学原理：
        M(q) = Σ [m_i × J_vi^T × J_vi + J_ωi^T × I_i × J_ωi]

        参数：
            q: 关节角度 (7,)

        返回：
            M: 7×7惯性矩阵
        """
        M = np.zeros((7, 7))

        # 计算正运动学，获取各关节位姿
        _, T_list = self.kin.forward_kinematics(q)

        for i in range(7):
            # 获取连杆i的位姿
            T_i = T_list[i + 1]
            R_i = T_i[:3, :3]  # 旋转矩阵
            p_i = T_i[:3, 3]   # 位置

            # 质心在世界坐标系中的位置
            p_ci = p_i + R_i @ self.com[i]

            # 计算质心的线速度雅可比 J_vi
            J_vi = self._compute_com_jacobian(T_list, i, p_ci)

            # 计算角速度雅可比 J_ωi
            J_omega = self._compute_angular_jacobian(T_list, i)

            # 惯性张量转换到世界坐标系
            I_world = R_i @ self.inertia[i] @ R_i.T

            # 累加到惯性矩阵
            M += self.mass[i] * (J_vi.T @ J_vi)
            M += J_omega.T @ I_world @ J_omega

        return M

    def _compute_com_jacobian(self, T_list, link_idx, p_com):
        """
        计算质心的线速度雅可比

        J_vi = [z0×(p_com-p0), z1×(p_com-p1), ..., z_{i}×(p_com-p_{i}), 0, ..., 0]
        """
        J = np.zeros((3, 7))

        for j in range(link_idx + 1):
            T_j = T_list[j]
            z_j = T_j[:3, 2]  # 关节j的旋转轴
            p_j = T_j[:3, 3]  # 关节j的位置

            J[:, j] = np.cross(z_j, p_com - p_j)

        return J

    def _compute_angular_jacobian(self, T_list, link_idx):
        """
        计算角速度雅可比

        J_ωi = [z0, z1, ..., z_i, 0, ..., 0]
        """
        J = np.zeros((3, 7))

        for j in range(link_idx + 1):
            T_j = T_list[j]
            z_j = T_j[:3, 2]
            J[:, j] = z_j

        return J
```

### 2.5 科里奥利力和离心力推导

```
科里奥利力矩阵的推导：

使用Christoffel符号：

c_ijk = 1/2 × (∂M_ij/∂q_k + ∂M_ik/∂q_j - ∂M_jk/∂q_i)

科里奥利力矩阵：
C_ij = Σ_k c_ijk × q̇_k

物理含义：
- 当关节运动时，由于惯性矩阵随构型变化，产生耦合力
- 科里奥利力：不同关节之间的耦合
- 离心力：单关节旋转产生的"向外"的力
```

```python
def compute_coriolis_matrix(self, q: np.ndarray, qd: np.ndarray) -> np.ndarray:
    """
    计算科里奥利力矩阵 C(q,q̇)

    数学原理：
    C_ij = Σ_k c_ijk × q̇_k
    c_ijk = 1/2 × (∂M_ij/∂q_k + ∂M_ik/∂q_j - ∂M_jk/∂q_i)

    参数：
        q: 关节角度 (7,)
        qd: 关节速度 (7,)

    返回：
        C: 7×7科里奥利力矩阵
    """
    C = np.zeros((7, 7))

    # 计算惯性矩阵的偏导数
    delta = 1e-6
    M = self.compute_mass_matrix(q)

    # 数值计算偏导数
    dM = np.zeros((7, 7, 7))  # dM[i][j][k] = ∂M_ij/∂q_k
    for k in range(7):
        q_plus = q.copy()
        q_plus[k] += delta
        M_plus = self.compute_mass_matrix(q_plus)
        dM[:, :, k] = (M_plus - M) / delta

    # 计算Christoffel符号和科里奥利矩阵
    for i in range(7):
        for j in range(7):
            for k in range(7):
                c_ijk = 0.5 * (dM[i, j, k] + dM[i, k, j] - dM[j, k, i])
                C[i, j] += c_ijk * qd[k]

    return C
```

### 2.6 重力项推导

```
重力项的推导：

势能：P = Σ m_i × g^T × p_ci

其中：
- g = [0, 0, -9.81]^T：重力加速度向量
- p_ci：连杆i质心在世界坐标系中的位置

重力项：G_i = ∂P/∂q_i

物理含义：
- 克服重力所需的关节力矩
- 与关节角度相关（因为质心位置随关节角变化）
```

```python
def compute_gravity_vector(self, q: np.ndarray) -> np.ndarray:
    """
    计算重力向量 G(q)

    数学原理：
    G_i = ∂P/∂q_i
    P = Σ m_i × g^T × p_ci

    参数：
        q: 关节角度 (7,)

    返回：
        G: 7×1重力向量
    """
    G = np.zeros(7)
    g = np.array([0, 0, -9.81])  # 重力加速度

    # 计算正运动学
    _, T_list = self.kin.forward_kinematics(q)

    for i in range(7):
        T_i = T_list[i + 1]
        R_i = T_i[:3, :3]
        p_i = T_i[:3, 3]

        # 质心位置
        p_ci = p_i + R_i @ self.com[i]

        # 质心的线速度雅可比
        J_vi = self._compute_com_jacobian(T_list, i, p_ci)

        # 势能对关节角的偏导数
        # ∂P/∂q_j = m_i × g^T × J_vi(:,j)
        G += self.mass[i] * (J_vi.T @ g)

    return G
```

### 2.7 完整的动力学方程

```python
def compute_dynamics(self, q, qd, qdd):
    """
    计算完整的动力学方程

    τ = M(q)q̈ + C(q,q̇)q̇ + G(q)

    参数：
        q: 关节角度 (7,)
        qd: 关节速度 (7,)
        qdd: 关节加速度 (7,)

    返回：
        tau: 关节力矩 (7,)
    """
    M = self.compute_mass_matrix(q)
    C = self.coriolis_matrix(q, qd)
    G = self.gravity_vector(q)

    tau = M @ qdd + C @ qd + G

    return tau

def forward_dynamics(self, q, qd, tau, tau_ext=None):
    """
    正向动力学：给定力矩，计算加速度

    M(q)q̈ = τ - C(q,q̇)q̇ - G(q) + τ_ext

    参数：
        q: 关节角度
        qd: 关节速度
        tau: 关节力矩
        tau_ext: 外部力矩

    返回：
        qdd: 关节加速度
    """
    M = self.compute_mass_matrix(q)
    C = self.coriolis_matrix(q, qd)
    G = self.gravity_vector(q)

    if tau_ext is None:
        tau_ext = np.zeros(7)

    rhs = tau - C @ qd - G + tau_ext
    qdd = np.linalg.solve(M, rhs)

    return qdd

def inverse_dynamics(self, q, qd, qdd):
    """
    逆向动力学：给定运动，计算所需力矩

    τ = M(q)q̈ + C(q,q̇)q̇ + G(q)

    参数：
        q: 关节角度
        qd: 关节速度
        qdd: 关节加速度

    返回：
        tau: 所需关节力矩
    """
    M = self.compute_mass_matrix(q)
    C = self.coriolis_matrix(q, qd)
    G = self.gravity_vector(q)

    tau = M @ qdd + C @ qd + G

    return tau
```

## 3. Newton-Euler递推算法

### 3.1 算法概述

```
Newton-Euler方法特点：
- 递推计算，效率高 O(n)
- 适合实时控制
- 不需要显式计算M, C, G

两步递推：
┌─────────────────────────────────────────────────────────────┐
│  向外递推（i = 0 → n-1）：                                   │
│  ────────────────────────                                    │
│  从基座到末端，计算各连杆的速度、加速度                       │
│                                                             │
│  输入：q, q̇, q̈                                              │
│  输出：各连杆的角速度ω、角加速度ω̇、线加速度a                  │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  向内递推（i = n-1 → 0）：                                   │
│  ────────────────────────                                    │
│  从末端到基座，计算各关节的力和力矩                           │
│                                                             │
│  输入：各连杆的运动学量                                      │
│  输出：关节力矩 τ                                            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 递推公式

```
向外递推（Forward Recursion）：

角速度：
ω_i = R_i^T × ω_{i-1} + θ̇_i × z_0

角加速度：
ω̇_i = R_i^T × ω̇_{i-1} + θ̈_i × z_0 + θ̇_i × (R_i^T × ω_{i-1}) × z_0

线加速度：
a_i = R_i^T × (a_{i-1} + ω̇_{i-1} × p_i + ω_{i-1} × (ω_{i-1} × p_i))

质心线加速度：
a_ci = a_i + ω̇_i × p_ci + ω_i × (ω_i × p_ci)

───────────────────────────────────────────────────────────────

向内递推（Backward Recursion）：

惯性力：
f_ci = m_i × a_ci

惯性力矩：
τ_ci = I_i × ω̇_i + ω_i × (I_i × ω_i)

关节力（向内传递）：
f_i = R_{i+1} × f_{i+1} + f_ci

关节力矩（向内传递）：
τ_i = τ_ci + R_{i+1} × τ_{i+1} + p_ci × f_ci + p_i × R_{i+1} × f_{i+1}

关节输出力矩：
τ_i^joint = τ_i^T × z_0
```

### 3.3 Newton-Euler实现

```python
def newton_euler_recursive(kinematics, link_params, q, qd, qdd):
    """
    Newton-Euler递推算法

    参数：
        kinematics: 运动学对象
        link_params: 连杆参数
        q: 关节角度 (7,)
        qd: 关节速度 (7,)
        qdd: 关节加速度 (7,)

    返回：
        tau: 关节力矩 (7,)
    """
    n = 7
    g = np.array([0, 0, 9.81])  # 重力加速度

    # 初始化
    omega = [np.zeros(3) for _ in range(n+1)]      # 角速度
    omega_dot = [np.zeros(3) for _ in range(n+1)]  # 角加速度
    a = [np.zeros(3) for _ in range(n+1)]          # 线加速度
    a[0] = -g  # 基座加速度（包含重力）

    # 向外递推
    for i in range(n):
        # 获取变换矩阵
        T_i = kinematics.dh_transform(q[i],
                                       kinematics.d[i],
                                       kinematics.a[i],
                                       kinematics.alpha[i])
        R_i = T_i[:3, :3]
        p_i = T_i[:3, 3]

        z0 = np.array([0, 0, 1])

        # 角速度
        omega[i+1] = R_i.T @ omega[i] + qd[i] * z0

        # 角加速度
        omega_dot[i+1] = (R_i.T @ omega_dot[i] +
                         qdd[i] * z0 +
                         qd[i] * np.cross(R_i.T @ omega[i], z0))

        # 线加速度
        a[i+1] = R_i.T @ (a[i] +
                          np.cross(omega_dot[i], p_i) +
                          np.cross(omega[i], np.cross(omega[i], p_i)))

    # 向内递推
    tau = np.zeros(n)
    f = [np.zeros(3) for _ in range(n+1)]
    tau_n = [np.zeros(3) for _ in range(n+1)]

    for i in range(n-1, -1, -1):
        T_i = kinematics.dh_transform(q[i],
                                       kinematics.d[i],
                                       kinematics.a[i],
                                       kinematics.alpha[i])
        R_i = T_i[:3, :3]
        p_i = T_i[:3, 3]

        m_i = link_params['mass'][i]
        I_i = link_params['inertia'][i]
        p_ci = link_params['com'][i]

        # 质心线加速度
        R_full = T_i[:3, :3]
        a_ci = (a[i+1] +
                np.cross(omega_dot[i+1], p_ci) +
                np.cross(omega[i+1], np.cross(omega[i+1], p_ci)))

        # 惯性力
        f_ci = m_i * a_ci

        # 惯性力矩
        I_world = R_full @ I_i @ R_full.T
        tau_ci = (I_world @ omega_dot[i+1] +
                  np.cross(omega[i+1], I_world @ omega[i+1]))

        # 向内传递
        if i < n-1:
            R_next = kinematics.dh_transform(q[i+1],
                                              kinematics.d[i+1],
                                              kinematics.a[i+1],
                                              kinematics.alpha[i+1])[:3, :3]
            f[i] = R_next @ f[i+1] + f_ci
            tau_n[i] = (tau_ci +
                       R_next @ tau_n[i+1] +
                       np.cross(p_ci, f_ci) +
                       np.cross(p_i, R_next @ f[i+1]))
        else:
            f[i] = f_ci
            tau_n[i] = tau_ci + np.cross(p_ci, f_ci)

        # 关节力矩
        tau[i] = tau_n[i] @ np.array([0, 0, 1])

    return tau
```

## 4. 动力学参数辨识

### 4.1 为什么需要参数辨识？

```
动力学参数的来源：

1. CAD模型
   - 优点：直接获取
   - 缺点：可能与实际不符（装配误差、线缆等）

2. 参数辨识
   - 优点：更准确
   - 缺点：需要实验

需要辨识的参数：
- 各连杆质量 m_i
- 各连杆惯性张量 I_i
- 各连杆质心位置 p_ci

对于7自由度臂：
- 每个连杆：1个质量 + 3个惯性矩 + 3个质心坐标 = 7个参数
- 7个连杆：7 × 7 = 49个参数
```

### 4.2 最小参数集

```
并非所有参数都是独立可辨识的

最小参数集（Base Parameters）：
- 将动力学方程重新参数化
- 只保留可辨识的参数组合

对于7自由度臂，最小参数集通常有 ~50-60个参数

辨识方法：
1. 时域辨识
   - 激励轨迹 + 测量力矩
   - 最小二乘法求解

2. 频域辨识
   - 频率响应分析
   - 适合线性化模型
```

### 4.3 激励轨迹设计

```
激励轨迹的目的：
- 激发所有动力学参数的影响
- 提高辨识精度

常用方法：
1. 傅里叶级数轨迹
   q(t) = q_0 + Σ [a_k × sin(kωt) + b_k × cos(kωt)]

2. 优化设计
   - 最小化参数估计的协方差
   - 满足关节限位和速度限制

示例：
def generate_exciting_trajectory(n_joints, n_harmonics=5, T=10, dt=0.001):
    """
    生成激励轨迹（傅里叶级数）
    """
    t = np.arange(0, T, dt)
    n_points = len(t)

    q = np.zeros((n_points, n_joints))
    qd = np.zeros((n_points, n_joints))
    qdd = np.zeros((n_points, n_joints))

    omega_f = 2 * np.pi / T  # 基频

    for i in range(n_joints):
        for k in range(1, n_harmonics + 1):
            # 随机幅度和相位
            a_k = np.random.uniform(-0.1, 0.1)
            b_k = np.random.uniform(-0.1, 0.1)

            q[:, i] += a_k * np.sin(k * omega_f * t) + b_k * np.cos(k * omega_f * t)
            qd[:, i] += k * omega_f * (a_k * np.cos(k * omega_f * t) - b_k * np.sin(k * omega_f * t))
            qdd[:, i] += -(k * omega_f)**2 * (a_k * np.sin(k * omega_f * t) + b_k * np.cos(k * omega_f * t))

    return q, qd, qdd
```

### 5.2 PID控制器

```python
class JointPIDController:
    """
    关节空间PID控制器

    控制律：
    τ = Kp × e + Ki × ∫e dt + Kd × ė

    其中：
    - e = q_desired - q_actual（位置误差）
    - ė = q̇_desired - q̇_actual（速度误差）
    """

    def __init__(self, kp, ki, kd, dt, integral_limit=10.0):
        """
        参数：
            kp: 比例增益 (7,) 或 标量
            ki: 积分增益
            kd: 微分增益
            dt: 控制周期
            integral_limit: 积分限幅
        """
        self.kp = np.atleast_1d(kp)
        self.ki = np.atleast_1d(ki)
        self.kd = np.atleast_1d(kd)
        self.dt = dt
        self.integral_limit = integral_limit

        self.integral = np.zeros(7)
        self.prev_error = np.zeros(7)

    def compute(self, q_desired, q_actual, qd_desired=None, qd_actual=None):
        """
        计算控制力矩

        参数：
            q_desired: 期望关节角 (7,)
            q_actual: 实际关节角 (7,)
            qd_desired: 期望关节速度 (7,)，可选
            qd_actual: 实际关节速度 (7,)，可选

        返回：
            tau: 控制力矩 (7,)
        """
        # 位置误差
        error = q_desired - q_actual

        # 积分项（带限幅）
        self.integral += error * self.dt
        self.integral = np.clip(self.integral,
                                -self.integral_limit,
                                self.integral_limit)

        # 微分项
        if qd_actual is not None:
            if qd_desired is not None:
                derivative = qd_desired - qd_actual
            else:
                derivative = -qd_actual
        else:
            derivative = (error - self.prev_error) / self.dt

        self.prev_error = error

        # PID输出
        tau = (self.kp * error +
               self.ki * self.integral +
               self.kd * derivative)

        return tau

    def reset(self):
        """重置控制器状态"""
        self.integral = np.zeros(7)
        self.prev_error = np.zeros(7)
```

### 5.3 计算力矩控制（Computed Torque Control）

```
计算力矩控制原理：

目标：实现精确的轨迹跟踪

控制律：
τ = M(q) × (q̈_d + Kd × ė + Kp × e) + C(q,q̇) × q̇ + G(q)

其中：
- e = q_d - q（位置误差）
- ė = q̇_d - q̇（速度误差）
- Kp, Kd：增益矩阵

原理：
1. 使用动力学模型进行反馈线性化
2. 将非线性系统转化为线性系统
3. 然后使用PD控制

优点：
- 精确轨迹跟踪
- 可以设置期望的闭环动态

缺点：
- 需要精确的动力学模型
- 对模型误差敏感
```

```python
class ComputedTorqueController:
    """
    计算力矩控制器

    控制律：
    τ = M(q) × (q̈_d + Kd × ė + Kp × e) + C(q,q̇) × q̇ + G(q)
    """

    def __init__(self, dynamics_model, kp, kd):
        """
        参数：
            dynamics_model: 动力学模型对象
            kp: 比例增益 (7,) 或 标量
            kd: 微分增益
        """
        self.model = dynamics_model
        self.kp = np.atleast_1d(kp)
        self.kd = np.atleast_1d(kd)

    def compute(self, q_desired, qd_desired, qdd_desired,
                q_actual, qd_actual):
        """
        计算控制力矩

        参数：
            q_desired: 期望关节角 (7,)
            qd_desired: 期望关节速度 (7,)
            qdd_desired: 期望关节加速度 (7,)
            q_actual: 实际关节角 (7,)
            qd_actual: 实际关节速度 (7,)

        返回：
            tau: 控制力矩 (7,)
        """
        # 计算误差
        e = q_desired - q_actual
        ed = qd_desired - qd_actual

        # 计算动力学项
        M = self.model.compute_mass_matrix(q_actual)
        C = self.model.compute_coriolis_matrix(q_actual, qd_actual)
        G = self.model.gravity_vector(q_actual)

        # 计算控制力矩
        # τ = M × (q̈_d + Kd × ė + Kp × e) + C × q̇ + G
        tau = M @ (qdd_desired + self.kd * ed + self.kp * e) + C @ qd_actual + G

        return tau
```

### 5.4 阻抗控制

```
阻抗控制原理：

目标：将机械臂末端表现为弹簧-阻尼系统

目标动力学：
M_d × (ẍ - ẍ_d) + D_d × (ẋ - ẋ_d) + K_d × (x - x_d) = F_ext

其中：
- M_d：期望惯量
- D_d：期望阻尼
- K_d：期望刚度
- F_ext：外部力

物理含义：
- K_d：刚度（抵抗位置偏差）
- D_d：阻尼（抵抗速度）
- M_d：惯量（抵抗加速度）

应用：
- 装配任务
- 力控制
- 人机协作
```

```python
class ImpedanceController:
    """
    阻抗控制器

    将末端表现为弹簧-阻尼系统
    """

    def __init__(self, dynamics_model, jacobian_func,
                 md=None, dd=None, kd=None):
        """
        参数：
            dynamics_model: 动力学模型
            jacobian_func: 雅可比计算函数
            md: 期望惯量矩阵 (6,6)
            dd: 期望阻尼矩阵 (6,6)
            kd: 期望刚度矩阵 (6,6)
        """
        self.model = dynamics_model
        self.jacobian_func = jacobian_func

        # 默认参数
        self.Md = md if md is not None else np.diag([1, 1, 1, 0.1, 0.1, 0.1])
        self.Dd = dd if dd is not None else np.diag([50, 50, 50, 5, 5, 5])
        self.Kd = kd if kd is not None else np.diag([1000, 1000, 1000, 100, 100, 100])

    def compute(self, q, qd, x_desired, xd_desired, xdd_desired,
                x_actual, xd_actual, f_ext=None):
        """
        计算关节力矩

        参数：
            q, qd: 当前关节状态
            x_desired: 期望末端位姿 (6,)
            xd_desired: 期望末端速度 (6,)
            xdd_desired: 期望末端加速度 (6,)
            x_actual: 实际末端位姿 (6,)
            xd_actual: 实际末端速度 (6,)
            f_ext: 外部力/力矩 (6,)

        返回：
            tau: 关节力矩 (7,)
        """
        if f_ext is None:
            f_ext = np.zeros(6)

        # 计算雅可比
        J = self.jacobian_func(q)

        # 笛卡尔空间误差
        e_x = x_desired - x_actual
        e_xd = xd_desired - xd_actual

        # 阻抗控制律
        # F = M_d × ẍ_d + D_d × ė_x + K_d × e_x + F_ext
        F = (self.Md @ xdd_desired +
             self.Dd @ e_xd +
             self.Kd @ e_x +
             f_ext)

        # 转换为关节空间力矩
        tau = J.T @ F

        # 加上重力补偿
        G = self.model.gravity_vector(q)
        tau += G

        return tau
```

```python
        p.loadURDF("plane.urdf")

        # 加载机器人
        self.robot_id = p.loadURDF(urdf_path, [0, 0, 0])

        # 获取关节信息
        self.num_joints = p.getNumJoints(self.robot_id)
        self.joint_info = self._get_joint_info()

        print(f"加载机器人，共 {self.num_joints} 个关节")

    def _get_joint_info(self):
        """获取关节信息"""
        info = []
        for i in range(self.num_joints):
            joint_info = p.getJointInfo(self.robot_id, i)
            info.append({
                'name': joint_info[1].decode('utf-8'),
                'type': joint_info[2],
                'lower_limit': joint_info[8],
                'upper_limit': joint_info[9],
                'max_force': joint_info[10],
                'max_velocity': joint_info[11]
            })
        return info

    def set_joint_positions(self, target_positions, force=500):
        """
        设置关节位置（位置控制）

        参数：
            target_positions: 目标关节角度列表
            force: 最大力矩
        """
        for i in range(min(len(target_positions), self.num_joints)):
            p.setJointMotorControl2(
                self.robot_id,
                i,
                p.POSITION_CONTROL,
                targetPosition=target_positions[i],
                force=force
            )

    def set_joint_torques(self, torques):
        """
        设置关节力矩（力矩控制）

        参数：
            torques: 关节力矩列表
        """
        for i in range(min(len(torques), self.num_joints)):
            p.setJointMotorControl2(
                self.robot_id,
                i,
                p.TORQUE_CONTROL,
                force=torques[i]
            )

    def get_joint_states(self):
        """
        获取关节状态

        返回：
            positions: 关节角度
            velocities: 关节速度
            torques: 关节力矩
        """
        states = p.getJointStates(self.robot_id, range(self.num_joints))

        positions = np.array([s[0] for s in states])
        velocities = np.array([s[1] for s in states])
        torques = np.array([s[3] for s in states])

        return positions, velocities, torques

    def get_link_pose(self, link_index):
        """
        获取连杆位姿

        返回：
            position: [x, y, z]
            orientation: [x, y, z, w] 四元数
        """
        state = p.getLinkState(self.robot_id, link_index)
        return np.array(state[0]), np.array(state[1])

    def step_simulation(self, dt=1/240):
        """执行一步仿真"""
        p.stepSimulation()
        time.sleep(dt)

    def run_trajectory(self, trajectory, dt=1/240, use_position_control=True):
        """
        执行轨迹

        参数：
            trajectory: 关节角度轨迹列表
            dt: 时间步长
            use_position_control: 是否使用位置控制
        """
        for q in trajectory:
            if use_position_control:
                self.set_joint_positions(q)
            self.step_simulation(dt)

    def close(self):
        """关闭仿真"""
        p.disconnect()
```

```python
        self.command_pub = self.create_publisher(
            Float64MultiArray,
            '/arm_controller/commands',
            10

        # PID参数
        self.kp = np.array([100, 100, 100, 50, 50, 50, 20])
        self.kd = np.array([10, 10, 10, 5, 5, 5, 2])

        # 控制定时器
        self.timer = self.create_timer(0.001, self.control_callback)  # 1kHz

        self.current_positions = np.zeros(7)
        self.current_velocities = np.zeros(7)

    def joint_state_callback(self, msg):
        """接收关节状态"""
        self.current_positions = np.array(msg.position)
        self.current_velocities = np.array(msg.velocity)

    def control_callback(self):
        """控制回调"""
        # 计算误差
        error = self.target_positions - self.current_positions

        # PD控制
        torques = self.kp * error - self.kd * self.current_velocities

        # 发布力矩命令
        msg = Float64MultiArray()
        msg.data = torques.tolist()
        self.command_pub.publish(msg)

    def set_target(self, positions):
        """设置目标位置"""
        self.target_positions = np.array(positions)
```

```python
    dynamics = DynamicsModel(kin, link_params)
    controller = ComputedTorqueController(dynamics, kp=100, kd=20)

    # 生成轨迹
    q_start = np.zeros(7)
    q_end = np.array([0, 0.5, 0, -0.5, 0, 0.3, 0])
    T = 2.0  # 运动时间
    dt = 0.001  # 控制周期

    # 五次多项式轨迹
    n_steps = int(T / dt)
    q_desired = np.zeros((n_steps, 7))
    qd_desired = np.zeros((n_steps, 7))
    qdd_desired = np.zeros((n_steps, 7))

    for i in range(7):
        dq = q_end[i] - q_start[i]
        a0 = q_start[i]
        a3 = 20 * dq / (2 * T**3)
        a4 = -30 * dq / (2 * T**4)
        a5 = 12 * dq / (2 * T**5)

        t = np.linspace(0, T, n_steps)
        q_desired[:, i] = a0 + a3*t**3 + a4*t**4 + a5*t**5
        qd_desired[:, i] = 3*a3*t**2 + 4*a4*t**3 + 5*a5*t**4
        qdd_desired[:, i] = 6*a3*t + 12*a4*t**2 + 20*a5*t**3

    # 仿真
    q_actual = np.zeros((n_steps, 7))
    qd_actual = np.zeros((n_steps, 7))
    tau_history = np.zeros((n_steps, 7))

    q = q_start.copy()
    qd = np.zeros(7)

    for step in range(n_steps):
        # 计算控制力矩
        tau = controller.compute(
            q_desired[step], qd_desired[step], qdd_desired[step],
            q, qd
        )

        # 正向动力学（仿真）
        qdd = dynamics.forward_dynamics(q, qd, tau)

        # 积分
        qd = qd + qdd * dt
        q = q + qd * dt

        # 记录
        q_actual[step] = q
        qd_actual[step] = qd
        tau_history[step] = tau

    # 绘图
    import matplotlib.pyplot as plt

    fig, axes = plt.subplots(3, 1, figsize=(12, 10))

    t = np.linspace(0, T, n_steps)

    # 关节角度
    for i in range(7):
        axes[0].plot(t, np.degrees(q_desired[:, i]), '--', alpha=0.5)
        axes[0].plot(t, np.degrees(q_actual[:, i]))
    axes[0].set_ylabel('角度 (deg)')
    axes[0].set_title('轨迹跟踪结果')
    axes[0].grid(True)

    # 跟踪误差
    error = np.linalg.norm(q_desired - q_actual, axis=1)
    axes[1].plot(t, error)
    axes[1].set_ylabel('跟踪误差 (rad)')
    axes[1].grid(True)

    # 控制力矩
    for i in range(7):
        axes[2].plot(t, tau_history[:, i])
    axes[2].set_ylabel('力矩 (Nm)')
    axes[2].set_xlabel('时间 (s)')
    axes[2].grid(True)

    plt.tight_layout()
    plt.savefig('trajectory_tracking.png', dpi=150)
    plt.show()
```

## 9. 常见问题

### Q1: 动力学模型验证怎么做？

**A**:
1. **自由落体仿真**：在重力下观察关节运动是否合理
2. **能量守恒**：无外力时总能量应守恒
3. **与已知模型对比**：如MATLAB Robotics Toolbox
4. **实验验证**：与实际机器人对比

### Q2: 计算力矩控制为什么需要精确模型？

**A**: 计算力矩控制依赖模型进行反馈线性化，模型误差会直接转化为跟踪误差。实践中常加鲁棒项或自适应项来补偿。

### Q3: PID和计算力矩控制怎么选？

**A**:
- **PID**：简单、不需要模型、适合慢速运动
- **计算力矩**：需要模型、精度高、适合快速运动

### Q4: 如何获取准确的动力学参数？

**A**:
1. **CAD模型**：从SolidWorks/Fusion 360导出
2. **参数辨识**：实验测量
3. **文献参考**：类似机器人的参数

### Q5: 仿真步长怎么设置？

**A**:
- 典型值：0.001s (1kHz) 到 0.01s (100Hz)
- 精度要求高：用更小步长
- 实时性要求：用更大步长
