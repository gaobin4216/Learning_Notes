## 1. 运动学基础概念

### 1.1 运动学的两大问题

```
┌─────────────────────────────────────────────────────────────────┐
│                        运动学两大问题                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  问题1：正运动学（Forward Kinematics, FK）                       │
│  ─────────────────────────────────────────                       │
│  已知：各关节角度 q = [θ1, θ2, ..., θ7]                         │
│  求解：末端执行器的位姿 T = [x, y, z, roll, pitch, yaw]         │
│                                                                 │
│  数学表示：T = f(q)                                              │
│                                                                 │
│  特点：解唯一，计算简单                                          │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  问题2：逆运动学（Inverse Kinematics, IK）                       │
│  ─────────────────────────────────────────                       │
│  已知：末端执行器的目标位姿 T*                                   │
│  求解：各关节角度 q = [θ1, θ2, ..., θ7]                         │
│                                                                 │
│  数学表示：q = f^(-1)(T*)                                        │
│                                                                 │
│  特点：对于7自由度有无穷多解，计算复杂                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 齐次变换矩阵

```
齐次变换矩阵（Homogeneous Transformation Matrix）

用于描述刚体在空间中的位姿（位置+姿态）

T = ┌ R   p ┐   =   ┌ r11  r12  r13  px ┐
    │       │        │ r21  r22  r23  py │
    │ 0   1 │        │ r31  r32  r33  pz │
    └       ┘        └  0    0    0    1  ┘

其中：
- R（3×3旋转矩阵）：描述姿态
- p（3×1向量）：描述位置
- 最后一行 [0 0 0 1]：齐次坐标

旋转矩阵R的性质：
1. R^T × R = I（正交性）
2. det(R) = 1（保体积）
3. R^(-1) = R^T
```

### 1.3 旋转的表示方法

```
1. 旋转矩阵（3×3）
   优点：无奇异性
   缺点：9个参数，有6个约束

2. 欧拉角（ZYX / ZYZ等）
   优点：直观，3个参数
   缺点：万向节锁（Gimbal Lock）

3. 四元数 q = [w, x, y, z]
   优点：无奇异性，计算高效
   缺点：不直观

4. 轴角（Axis-Angle）
   优点：直观
   缺点：表示不唯一

转换关系：
┌─────────────────────────────────────────────────────────────┐
│  旋转矩阵 ←→ 欧拉角（ZYX顺序，roll-pitch-yaw）             │
│                                                             │
│  R = Rz(yaw) × Ry(pitch) × Rx(roll)                        │
│                                                             │
│  R = ┌ cos(y)cos(p)  cos(y)sin(p)sin(r)-sin(y)cos(r) ... ┐│
│      │ sin(y)cos(p)  sin(y)sin(p)sin(r)+cos(y)cos(r) ... ││
│      │ -sin(p)       cos(p)sin(r)                 ...     ││
│      └                                                   ┘ │
│                                                             │
│  反解：                                                     │
│  pitch = atan2(-r31, sqrt(r32² + r33²))                    │
│  roll  = atan2(r32/cos(p), r33/cos(p))                     │
│  yaw   = atan2(r21/cos(p), r11/cos(p))                     │
└─────────────────────────────────────────────────────────────┘
```

## 2. 正运动学详解

### 2.1 齐次变换矩阵的链式法则

```
正运动学的计算过程：

T_0n = T_01 × T_12 × T_23 × ... × T_(n-1)n

对于7自由度机械臂：
T_07 = T_01 × T_12 × T_23 × T_34 × T_45 × T_56 × T_67

其中 T_i(i+1) 是第i+1个关节相对于第i个坐标系的变换矩阵

几何解释：
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  基座 ──T01──> 关节1 ──T12──> 关节2 ──T23──> ... ──T67──> 末端│
│                                                             │
│  每一步变换都相对于前一个坐标系                               │
│  最终结果是末端相对于基座的位姿                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 完整的正运动学代码实现

```python
import numpy as np
from typing import List, Tuple

class SevenDOFKinematics:
    """七自由度机械臂运动学类"""

    def __init__(self, dh_params: dict):
        """
        初始化

        参数：
            dh_params: DH参数字典
                d: 各关节连杆偏移 [d1, d2, ..., d7] (m)
                a: 各关节连杆长度 [a1, a2, ..., a7] (m)
                alpha: 各关节连杆扭转角 [α1, α2, ..., α7] (rad)
        """
        self.d = np.array(dh_params['d'])
        self.a = np.array(dh_params['a'])
        self.alpha = np.array(dh_params['alpha'])

        # 关节限位（默认±180度）
        self.joint_limits = np.array([
            [-np.pi, np.pi],
            [-np.pi/2, np.pi/2],
            [-np.pi, np.pi],
            [-np.pi/2, np.pi/2],
            [-np.pi, np.pi],
            [-np.pi/2, np.pi/2],
            [-np.pi, np.pi]
        ])

    def dh_transform(self, theta: float, d: float, a: float, alpha: float) -> np.ndarray:
        """
        计算单关节齐次变换矩阵

        参数：
            theta: 关节角（rad）
            d: 连杆偏移（m）
            a: 连杆长度（m）
            alpha: 连杆扭转角（rad）

        返回：
            T: 4×4齐次变换矩阵

        公式：
            T = ┌ cosθ  -sinθ·cosα   sinθ·sinα   a·cosθ ┐
                │ sinθ   cosθ·cosα  -cosθ·sinα   a·sinθ │
                │  0       sinα        cosα         d    │
                └  0         0           0          1    ┘
        """
        ct = np.cos(theta)
        st = np.sin(theta)
        ca = np.cos(alpha)
        sa = np.sin(alpha)

        T = np.array([
            [ct, -st*ca,  st*sa, a*ct],
            [st,  ct*ca, -ct*sa, a*st],
            [0,   sa,     ca,    d   ],
            [0,   0,      0,     1   ]
        ])

        return T

    def forward_kinematics(self, q: np.ndarray) -> Tuple[np.ndarray, List[np.ndarray]]:
        """
        正运动学计算

        参数：
            q: 关节角度 [θ1, θ2, ..., θ7] (rad)

        返回：
            T: 末端执行器的4×4齐次变换矩阵
            T_list: 各关节的变换矩阵列表（用于可视化）

        计算过程：
            T_07 = T_01 × T_12 × T_23 × T_34 × T_45 × T_56 × T_67
        """
        T = np.eye(4)
        T_list = [T.copy()]  # 基座坐标系

        for i in range(7):
            Ti = self.dh_transform(q[i], self.d[i], self.a[i], self.alpha[i])
            T = T @ Ti
            T_list.append(T.copy())

        return T, T_list

    def get_end_effector_pose(self, q: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """
        获取末端执行器的位姿

        参数：
            q: 关节角度

        返回：
            position: [x, y, z] 位置（m）
            rpy: [roll, pitch, yaw] 姿态（rad）
        """
        T, _ = self.forward_kinematics(q)

        # 提取位置
        position = T[:3, 3]

        # 提取RPY角（ZYX顺序）
        roll = np.arctan2(T[2, 1], T[2, 2])
        pitch = np.arctan2(-T[2, 0], np.sqrt(T[2, 1]**2 + T[2, 2]**2))
        yaw = np.arctan2(T[1, 0], T[0, 0])

        rpy = np.array([roll, pitch, yaw])

        return position, rpy

    def get_joint_positions(self, q: np.ndarray) -> List[np.ndarray]:
        """
        获取各关节的位置（用于可视化）

        返回：
            positions: 各关节位置列表
        """
        _, T_list = self.forward_kinematics(q)
        positions = [T[:3, 3] for T in T_list]
        return positions


# 使用示例
if __name__ == "__main__":
    # 定义DH参数（桌面级七自由度臂）
    dh_params = {
        'd': [0.34, 0, 0.30, 0, 0.30, 0, 0.06],
        'a': [0, 0.04, 0, 0, 0, 0, 0],
        'alpha': [-np.pi/2, np.pi/2, -np.pi/2, np.pi/2, -np.pi/2, np.pi/2, 0]
    }

    # 创建运动学对象
```

## 3. 雅可比矩阵推导

### 3.1 雅可比矩阵的定义

```
雅可比矩阵（Jacobian Matrix）描述关节速度到末端速度的映射

定义：
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  v = J(q) × q̇                                               │
│                                                             │
│  其中：                                                      │
│  - v (6×1)：末端速度 [vx, vy, vz, ωx, ωy, ωz]              │
│  - J(q) (6×7)：雅可比矩阵                                   │
│  - q̇ (7×1)：关节速度 [θ̇1, θ̇2, ..., θ̇7]                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘

雅可比矩阵的物理意义：
- J的第i列表示第i个关节单位速度对末端速度的贡献
- J将关节空间映射到操作空间
```

### 3.2 雅可比矩阵的几何推导

```
对于旋转关节i，雅可比矩阵的第i列：

┌        ┐   ┌           ┐
│  Jvi   │   │ z_(i-1) × p_e | │
│        │ = │                 │
│  Jωi   │   │    z_(i-1)     │
└        ┘   └               ┘

其中：
- z_(i-1)：关节i的旋转轴（在基座坐标系中表示）
- p_e：末端位置向量
- p_(i-1)：关节i的位置向量
- ×：叉乘

展开：
Jvi = z_(i-1) × (p_e - p_(i-1))
Jωi = z_(i-1)
```

### 3.3 完整的雅可比矩阵计算

```python
def compute_jacobian(self, q: np.ndarray) -> np.ndarray:
    """
    计算雅可比矩阵（几何法）

    参数：
        q: 关节角度

    返回：
        J: 6×7雅可比矩阵

    推导：
        对于旋转关节i：
        J_i = [z_{i-1} × (p_e - p_{i-1})]  (线速度部分)
              [        z_{i-1}         ]  (角速度部分)
    """
    # 计算正运动学，获取各关节位姿
    T_ee, T_list = self.forward_kinematics(q)

    # 末端位置
    p_e = T_ee[:3, 3]

    # 初始化雅可比矩阵
    J = np.zeros((6, 7))

    for i in range(7):
        # 第i个关节的坐标系
        T_i = T_list[i]

        # 关节i的旋转轴（Z轴方向，在基座坐标系中）
        z_i = T_i[:3, 2]

        # 关节i的位置
        p_i = T_i[:3, 3]

        # 线速度部分：Jv_i = z_i × (p_e - p_i)
        J[:3, i] = np.cross(z_i, p_e - p_i)

        # 角速度部分：Jw_i = z_i
        J[3:, i] = z_i

    return J


def compute_jacobian_numerical(self, q: np.ndarray, delta: float = 1e-6) -> np.ndarray:
    """
    数值法计算雅可比矩阵（用于验证）

    参数：
        q: 关节角度
        delta: 数值微分步长

    返回：
        J: 6×7雅可比矩阵
    """
    # 当前位姿
    T_current, _ = self.forward_kinematics(q)
    x_current = np.concatenate([
        T_current[:3, 3],
        self._rotation_to_rpy(T_current[:3, :3])
    ])

    J = np.zeros((6, 7))

    for i in range(7):
        # 对第i个关节做微小变化
        q_delta = q.copy()
        q_delta[i] += delta

        # 计算变化后的位姿
        T_delta, _ = self.forward_kinematics(q_delta)
        x_delta = np.concatenate([
            T_delta[:3, 3],
            self._rotation_to_rpy(T_delta[:3, :3])
        ])

        # 数值微分
        J[:, i] = (x_delta - x_current) / delta

    return J

def _rotation_to_rpy(self, R: np.ndarray) -> np.ndarray:
    """旋转矩阵转RPY角"""
    roll = np.arctan2(R[2, 1], R[2, 2])
    pitch = np.arctan2(-R[2, 0], np.sqrt(R[2, 1]**2 + R[2, 2]**2))
    yaw = np.arctan2(R[1, 0], R[0, 0])
    return np.array([roll, pitch, yaw])
```

### 3.4 雅可比矩阵的性质

```
1. 速度映射
   v = J × q̇
   末端速度 = 雅可比 × 关节速度

2. 力映射（虚功原理）
   τ = J^T × F
   关节力矩 = 雅可比转置 × 末端力

3. 奇异性
   当 det(J × J^T) → 0 时，雅可比奇异
   此时某些方向无法运动

4. 可操作性
   w = sqrt(det(J × J^T))
   可操作性越大，灵巧性越好
```

## 4. 逆运动学求解

### 4.1 七自由度逆运动学的挑战

```
问题特点：

1. 冗余自由度
   - 7个未知数，6个方程
   - 欠定方程组，有无穷多解
   - 需要额外约束选择最优解

2. 非线性
   - 无法用线性代数直接求解
   - 需要迭代方法

3. 奇异点
   - 某些构型下雅可比矩阵退化
   - 数值方法可能失败

解决策略：
┌─────────────────────────────────────────────────────────────┐
│  方法1：雅可比伪逆法                                         │
│  Δθ = J† × Δx                                              │
│  优点：简单直观                                              │
│  缺点：奇异点附近不稳定                                      │
├─────────────────────────────────────────────────────────────┤
│  方法2：阻尼最小二乘法（DLS）                                │
│  Δθ = J^T × (J × J^T + λ²I)^(-1) × Δx                     │
│  优点：奇异点附近稳定                                        │
│  缺点：精度略低                                              │
├─────────────────────────────────────────────────────────────┤
│  方法3：零空间优化                                           │
│  Δθ = J† × Δx + (I - J† × J) × q0                         │
│  优点：可以优化额外目标                                      │
│  缺点：需要定义优化目标                                      │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 雅可比伪逆法详解

```
算法原理：

目标：找到关节角变化 Δθ，使得末端位姿误差最小

线性近似：
Δx ≈ J × Δθ

求解（最小范数解）：
Δθ = J† × Δx = J^T × (J × J^T)^(-1) × Δx

其中 J† 是雅可比的伪逆

迭代过程：
┌─────────────────────────────────────────────────────────────┐
│  初始化：q = q_initial                                       │
│                                                             │
│  循环：                                                      │
│    1. 计算当前位姿 x_current = FK(q)                        │
│    2. 计算误差 Δx = x_target - x_current                   │
│    3. 如果 ||Δx|| < ε，退出                                 │
│    4. 计算雅可比 J = Jacobian(q)                            │
│    5. 计算伪逆 J† = pinv(J)                                 │
│    6. 更新关节角 q = q + α × J† × Δx                       │
│                                                             │
│  参数：                                                      │
│    - α：步长（通常0.1~1.0）                                  │
│    - ε：收敛阈值（通常1e-4~1e-6）                            │
│    - 最大迭代次数：100~1000                                  │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 完整的逆运动学实现

```python
class SevenDOFIK:
    """七自由度逆运动学求解器"""

    def __init__(self, kinematics: SevenDOFKinematics):
        """
        参数：
            kinematics: 正运动学对象
        """
        self.kin = kinematics

    def inverse_kinematics_jacobian(
        self,
        target_pose: np.ndarray,
        initial_angles: np.ndarray,
        max_iter: int = 100,
        tol: float = 1e-4,
        alpha: float = 0.1
    ) -> Tuple[np.ndarray, bool]:
        """
        雅可比伪逆法求解逆运动学

        参数：
            target_pose: 目标位姿 [x, y, z, roll, pitch, yaw]
            initial_angles: 初始关节角猜测
            max_iter: 最大迭代次数
            tol: 收敛阈值
            alpha: 步长

        返回：
            q_sol: 求解得到的关节角
            success: 是否收敛
        """
        q = initial_angles.copy()

        for iteration in range(max_iter):
            # 当前位姿
            T_current, _ = self.kin.forward_kinematics(q)
            pos_current = T_current[:3, 3]
            rpy_current = self.kin._rotation_to_rpy(T_current[:3, :3])
            x_current = np.concatenate([pos_current, rpy_current])

            # 位姿误差
            error = target_pose - x_current

            # 检查收敛
            if np.linalg.norm(error) < tol:
                print(f"收敛于第 {iteration} 次迭代")
                return q, True

            # 计算雅可比矩阵
            J = self.kin.compute_jacobian(q)

            # 伪逆法更新
            J_pinv = np.linalg.pinv(J)
            delta_q = alpha * J_pinv @ error

            # 更新关节角
            q += delta_q

            # 关节角限位（可选）
            q = np.clip(q,
                       self.kin.joint_limits[:, 0],
                       self.kin.joint_limits[:, 1])

        print(f"未收敛，最终误差: {np.linalg.norm(error):.6f}")
        return q, False

    def inverse_kinematics_dls(
        self,
        target_pose: np.ndarray,
        initial_angles: np.ndarray,
        max_iter: int = 100,
        tol: float = 1e-4,
        alpha: float = 0.1,
        damping: float = 0.01
    ) -> Tuple[np.ndarray, bool]:
        """
        阻尼最小二乘法求解逆运动学

        参数：
            damping: 阻尼因子，避免奇异点附近数值不稳定

        数学原理：
            Δθ = J^T × (J × J^T + λ²I)^(-1) × Δx

        优点：
            - 在奇异点附近更稳定
            - 不会产生过大的关节速度
        """
        q = initial_angles.copy()

        for iteration in range(max_iter):
            T_current, _ = self.kin.forward_kinematics(q)
            pos_current = T_current[:3, 3]
            rpy_current = self.kin._rotation_to_rpy(T_current[:3, :3])
            x_current = np.concatenate([pos_current, rpy_current])

            error = target_pose - x_current

            if np.linalg.norm(error) < tol:
                print(f"收敛于第 {iteration} 次迭代")
                return q, True

            J = self.kin.compute_jacobian(q)

            # DLS公式
            Jt = J.T
            JtJ = J @ Jt
            damped_inv = np.linalg.solve(
                JtJ + (damping ** 2) * np.eye(6),
                error
            )
            delta_q = alpha * Jt @ damped_inv

            q += delta_q

        print(f"未收敛，最终误差: {np.linalg.norm(error):.6f}")
        return q, False

    def inverse_kinematics_redundant(
        self,
        target_pose: np.ndarray,
        initial_angles: np.ndarray,
        secondary_objective=None,
        max_iter: int = 100,
        tol: float = 1e-4
    ) -> Tuple[np.ndarray, bool]:
        """
        带冗余度优化的逆运动学

        数学原理：
            Δθ = J† × Δx + (I - J† × J) × q0

        其中：
            - J† × Δx：主任务（跟踪目标位姿）
            - (I - J† × J) × q0：零空间任务（优化额外目标）

        常用的次要目标：
            - 最小化关节运动
            - 避障
            - 最大化可操作性
            - 避免关节限位
        """
        q = initial_angles.copy()
        I7 = np.eye(7)

        if secondary_objective is None:
            # 默认：趋向初始位置
            secondary_objective = lambda q: 0.1 * (initial_angles - q)

        for iteration in range(max_iter):
            T_current, _ = self.kin.forward_kinematics(q)
            pos_current = T_current[:3, 3]
            rpy_current = self.kin._rotation_to_rpy(T_current[:3, :3])
            x_current = np.concatenate([pos_current, rpy_current])

            error = target_pose - x_current

            if np.linalg.norm(error) < tol:
                print(f"收敛于第 {iteration} 次迭代")
                return q, True

            J = self.kin.compute_jacobian(q)
            J_pinv = np.linalg.pinv(J)

            # 主任务
            primary = J_pinv @ error

            # 零空间投影矩阵
            null_space = I7 - J_pinv @ J

            # 次要任务
            secondary = secondary_objective(q)

            # 完整更新
            delta_q = primary + null_space @ secondary

            # 步长控制
            alpha = 0.1
            q += alpha * delta_q

        print(f"未收敛，最终误差: {np.linalg.norm(error):.6f}")
        return q, False
```

### 5.2 五次多项式轨迹规划

```
五次多项式（Quintic Polynomial）

位置：q(t) = a0 + a1*t + a2*t² + a3*t³ + a4*t⁴ + a5*t⁵
速度：q̇(t) = a1 + 2*a2*t + 3*a3*t² + 4*a4*t³ + 5*a5*t⁴
加速度：q̈(t) = 2*a2 + 6*a3*t + 12*a4*t² + 20*a5*t³

边界条件（6个方程，6个未知数）：
┌─────────────────────────────────────┐
│  q(0) = q_start                     │
│  q(T) = q_end                       │
│  q̇(0) = 0    (起始速度为0)          │
│  q̇(T) = 0    (终止速度为0)          │
│  q̈(0) = 0    (起始加速度为0)        │
│  q̈(T) = 0    (终止加速度为0)        │
└─────────────────────────────────────┘

求解系数：
a0 = q_start
a1 = 0
a2 = 0
a3 = 20*(q_end - q_start) / (2*T³)
a4 = -30*(q_end - q_start) / (2*T⁴)
a5 = 12*(q_end - q_start) / (2*T⁵)
```

```python
    ) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:

        五次多项式轨迹规划

        参数：
            q_start: 起始关节角 (7,)
            q_end: 目标关节角 (7,)
            T: 运动总时间 (s)

            a1 = 0
            a2 = 0
```

### 8.1 奇异点的类型

### 8.2 奇异点检测

```python
def check_singularity(self, q: np.ndarray) -> dict:
    """
    检测奇异点

    返回：
        dict: 包含可操作性、条件数等指标
    """
    J = self.compute_jacobian(q)

    # 可操作性（Manipulability）
    # w = sqrt(det(J × J^T))
    JJT = J @ J.T
    w = np.sqrt(np.abs(np.linalg.det(JJT)))

    # 条件数（Condition Number）
    # κ = σ_max / σ_min
    singular_values = np.linalg.svd(J, compute_uv=False)
    condition_number = singular_values[0] / singular_values[-1]

    # 最小奇异值
    min_singular_value = singular_values[-1]

    # 判断是否接近奇异
    is_singular = w < 0.01 or condition_number > 100

    return {
        'manipulability': w,
        'condition_number': condition_number,
        'min_singular_value': min_singular_value,
        'is_singular': is_singular
    }

def plot_manipulability_workspace(self, n_samples=1000):
    """
    绘制可操作性分布图
    """
    manipulabilities = []
    positions = []

    for _ in range(n_samples):
        # 随机关节角
        q = np.random.uniform(
            self.joint_limits[:, 0],
            self.joint_limits[:, 1]
        )

        # 计算可操作性
        result = self.check_singularity(q)
        manipulabilities.append(result['manipulability'])

        # 计算位置
        T, _ = self.forward_kinematics(q)
        positions.append(T[:3, 3])

    positions = np.array(positions)
    manipulabilities = np.array(manipulabilities)

    # 绘图
    fig = plt.figure(figsize=(10, 8))
    ax = fig.add_subplot(111, projection='3d')

    scatter = ax.scatter(
        positions[:, 0],
        positions[:, 1],
        positions[:, 2],
        c=manipulabilities,
        cmap='viridis',
        s=5
    )

    plt.colorbar(scatter, label='可操作性')
    ax.set_xlabel('X (m)')
    ax.set_ylabel('Y (m)')
    ax.set_zlabel('Z (m)')
    ax.set_title('可操作性分布图')
```

### 8.3 奇异点回避

```python
def inverse_kinematics_with_singularity_avoidance(
    self,
    target_pose: np.ndarray,
    initial_angles: np.ndarray,
    damping_factor: float = 0.01,
    max_iter: int = 100,
    tol: float = 1e-4
) -> Tuple[np.ndarray, bool]:
    """
    带奇异点回避的逆运动学

    使用动态调整的阻尼因子
    """
    q = initial_angles.copy()

    for iteration in range(max_iter):
        T_current, _ = self.forward_kinematics(q)
        x_current = np.concatenate([
            T_current[:3, 3],
            self._rotation_to_rpy(T_current[:3, :3])
        ])

        error = target_pose - x_current

        if np.linalg.norm(error) < tol:
            return q, True

        J = self.compute_jacobian(q)

        # 动态调整阻尼因子
        singular_values = np.linalg.svd(J, compute_uv=False)
        min_sv = singular_values[-1]

        if min_sv < 0.1:
            # 接近奇异，增大阻尼
            damping = damping_factor * (1 - min_sv / 0.1)
        else:
            damping = 0.001

        # DLS求解
        Jt = J.T
        JtJ = J @ Jt
        damped_inv = np.linalg.solve(
            JtJ + (damping ** 2) * np.eye(6),
            error
        )
        delta_q = 0.1 * Jt @ damped_inv

        q += delta_q

    return q, False
```

## 9. 常见问题

### Q1: 逆运动学求解不收敛怎么办？

**A**:
1. 检查目标位姿是否在工作空间内
2. 尝试不同的初始猜测
3. 减小步长（alpha参数）
4. 增加阻尼因子
5. 使用优化方法（如`ikcon`）

### Q2: 如何验证DH参数是否正确？

**A**:
1. **零位验证**：所有关节角为0，检查末端位姿
2. **单关节验证**：仅改变一个关节角，观察运动方向
3. **对比CAD**：与SolidWorks模型对比
4. **极限位置**：手臂完全伸展/收缩，检查距离

### Q3: 七自由度的冗余自由度有什么用？

**A**:
1. **避障**：保持末端位姿不变，调整中间关节
2. **避奇异**：远离奇异构型
3. **优化目标**：最小化能量、最大化可操作性等

### Q4: 轨迹规划时出现抖动怎么办？

**A**:
1. 使用高阶多项式（五次或更高）
2. 增加轨迹点密度
3. 检查逆运动学解的连续性
4. 使用关节空间插值代替笛卡尔空间

### Q5: 如何选择轨迹规划方法？

**A**:
- **关节空间规划**：简单、无奇异、适合点到点运动
- **笛卡尔空间规划**：末端轨迹直观、适合直线运动
- **五次多项式**：平滑、无冲击
- **梯形速度曲线**：简单、速度可控
