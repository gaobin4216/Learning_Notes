- **编程语言基础**：C++、Python、MATLAB、Java 的环境搭建与核心概念

- **应用开发**：从 Web 前后端到桌面应用（Qt / WinUI3）的全栈概览
- **开发工具**：Git 工作流、Markdown 速查、Vim 编辑等日常工具参考
- **AI与大模型**：LLM 核心概念与 AI 编程工具介绍
